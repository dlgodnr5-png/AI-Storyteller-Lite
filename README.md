## 프로젝트 개요

AI Storyteller Lite는 제목 선택 후 대본, TTS, 컷, 이미지/영상 훅, 자막, 최종 렌더, 발행 준비까지 이어지는 싱크 우선 자동 제작 도구입니다.

## 현재 반영된 핵심 기능 (패널 12 중심)

- 하이브리드 렌더: 초반 훅 컷은 영상, 이후 컷은 이미지 슬라이드로 혼합 렌더
- 영상 훅 컷 수 지정: Panel 12에서 영상 생성 컷 수 슬라이더로 제어
- BGM/SFX 미리듣기: Panel 12에서 배경음악/효과음 드롭다운 + 미리듣기 버튼 제공
- 템플릿 제목 오버레이 자동 작성: 템플릿 선택 시 제목 자동 재작성
- 수동 수정 보호: 제목을 수동 편집한 경우 템플릿 재클릭 전 자동 덮어쓰기 방지 안내
- 싱크 리포트: 대본/TTS/컷/렌더/SRT 마지막/오차(정상/주의/실패) 표시

## 규칙 반영 현황

### 1) 쇼츠 / 롱폼 대본 규칙
- 쇼츠는 KR/EN/JP 단위 제한 기반으로 자동 절삭 적용
- 쇼츠는 예상 재생시간(1.0x/1.25x)과 제한 초과 경고 표시
- 롱폼은 30초당 1컷 목표를 기준으로 컷 수 자동 재조정(압축/분할)

### 2) TTS 규칙
- 스타일 지시문 + 선택 음성 기반 TTS 생성
- 실측 길이 자동 측정(measuredDuration) 후 싱크 계산에 반영
- 상품 20초 모드에서 길이 초과 시 대본 절삭 후 TTS 재생성 보정

### 3) 영상/이미지 생성 규칙
- 컷별 이미지 자동 생성 배치(재시도 포함)
- 컷별 영상 업로드 지원(영상 훅 용도)
- 업로드 영상 메타 길이(videoDurationSec) 추출 후 렌더 타임라인 반영

### 4) 최종 싱크 규칙
- 렌더 타임라인은 이미지 고정 길이 + 영상 메타 길이를 함께 사용
- SRT 타임라인과 TTS 끝 시점을 비교해 오차(deltaSec) 계산
- 오차 기준: 정상 <= 0.25초 / 주의 <= 0.80초 / 실패 > 0.80초

## 원클릭 자동 제작 흐름

1. 대본 생성
2. 템플릿 제목 자동 재작성
3. TTS 생성
4. 컷 분할
5. 컷별 프롬프트 생성
6. 이미지 자동 생성
7. 슬라이드(영상 훅 + 이미지) 구성
8. 최종 렌더
9. 설명/태그 생성 및 발행 초안 채움

## 배포 확인 메모

운영 페이지에서 변경이 안 보일 경우:
1. 최신 커밋 푸시 여부 확인
2. 배포 파이프라인 완료 여부 확인
3. 브라우저 강력 새로고침(Ctrl+F5)
