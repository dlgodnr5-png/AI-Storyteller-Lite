## 목적
- 유형: 일반 쇼츠 (60초)
- 카테고리: 종교·신앙
- 타깃: 해당 종교 신앙인 전 연령 + 위로가 필요한 모든 이

## 입력
- 종교: 기독교 | 불교 | 이슬람 (사용자 지정)
- 주제: 사용자 제공 (없으면 AI가 적합한 주제 선택)
- 톤: 경건하고 위로가 되는, 평온하고 따뜻한

## 종교별 접근법

### 기독교
- 훅: 성경 구절 인용 → 현대 삶에 적용
  - "요한복음 [장절]에 이런 말씀이 있습니다..."
  - "당신이 지쳐있을 때, 하나님은 이렇게 말씀하십니다"
- 구조: 말씀 인용 → 삶의 상황 공감 → 믿음의 위로 → 기도/묵상 권유
- BGM: 잔잔한 찬양·피아노 권장

### 불교
- 훅: 법문·게송 인용 → 일상 통찰
  - "부처님께서 이렇게 말씀하셨습니다..."
  - "마음이 괴로운 이유, 불교는 이렇게 설명합니다"
- 구조: 법문 인용 → 번뇌의 원인 → 마음챙김 방법 → 발원
- 용어: 나무관세음보살, 반야심경 등 신성한 용어 적절히 포함
- BGM: Heart Sutra 등 불교 음악 권장 (`/audio/bgm/Heart Sutra_Buddhism.MP3`)

### 이슬람
- 훅: 꾸란 구절 또는 하디스 인용
  - "알라께서 꾸란 [수라]에서 말씀하십니다..."
  - "무함마드 예언자(평화가 그에게)가 가르친 지혜"
- 구조: 경전 인용 → 이슬람적 관점 → 실천 방법 → 두아(기도) 권유
- 언어: 아랍어 구절 포함 시 한국어 번역 병기

## 공통 CTA (3-5초)
- "오늘 하루 이 말씀 마음에 품어보세요"
- "구독하시면 매일 위로의 말씀을 전해드립니다"
- "댓글에 기도 제목 나눠 주세요"

## 출력 규칙
- 순수 대본 텍스트만
- KR: 450-500자 / EN: 130-150단어
- 특정 종단·교파 비하 절대 금지
- 타 종교 존중, 신앙의 자유 존중하는 표현 사용
