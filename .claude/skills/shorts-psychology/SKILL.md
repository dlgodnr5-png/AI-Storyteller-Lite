## 목적
- 유형: 일반 쇼츠 (60초)
- 카테고리: 심리학·자기계발
- 타깃: 20-40대 자기계발 관심층, 감정 관리·인간관계에 어려움을 겪는 직장인

## 입력
- 주제: 사용자 제공
- 타깃: 20-40대 직장인·학생 (기본값)
- 톤: 통찰+공감, 학술적 근거 + 친근한 설명

## 프로세스

### 훅 (3-5초) — 공감 또는 반전 통찰
- 공감 질문형: "당신이 항상 지쳐있는 진짜 이유, 알고 계신가요?"
- 반전 사실형: "착한 사람이 더 쉽게 번아웃되는 심리학적 이유"
- 행동 유발형: "지금 당장 SNS를 끊어야 하는 과학적 이유"
- 공식: [당신의 고민/증상] + [의외의 원인 또는 해결책]

### 본문 (48-50초) — 심리학적 근거 + 실용 적용
1. **현상 공감** (10초): 많은 사람이 겪는 구체적 상황 묘사
2. **심리학적 설명** (20초): 연구·이론·심리학 용어로 원인 설명
   - 인지 편향, 방어 기제, 애착 유형, 번아웃 등 구체적 개념 활용
3. **실천 방법** (20초): 오늘 바로 적용 가능한 1-2가지 구체적 방법

### CTA (3-5초) — 자기 적용 유도
- "오늘 딱 하나만 해보세요"
- "댓글에 어떤 방법이 도움됐는지 알려주세요"
- "이런 내용이 더 필요하시면 구독해 주세요"

## 출력 규칙
- 순수 대본 텍스트만
- KR: 450-500자 / EN: 130-150단어
- 심리학 용어 사용 시 쉬운 비유로 부연 설명
- 진단·치료 행위 금지, 전문 상담 권유 문구 필요 시 포함

## 검증된 고성과 주제
- 가스라이팅 / 나르시시즘 인식법
- 번아웃 예방법
- 도파민 디톡스
- 경계선 긋기 (바운더리)
- 내향인·외향인 특성
- 확증 편향 극복
- 자기 자신을 사랑하는 방법
