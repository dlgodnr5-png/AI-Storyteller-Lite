# 자막 템플릿 이미지 가이드 (1080x1920)

이 폴더의 이미지는 `12. 최종 영상생성 > 자막 템플릿` 카드의 미리보기로 사용됩니다.

- 권장 크기: `1080x1920` (9:16)
- 포맷: `png` 또는 `jpg`
- 앱은 카드에서 작은 이미지로 보여주고, 마우스 오버 시 확대 미리보기를 표시합니다.
- 템플릿 선택 시 아래 항목이 자동 적용됩니다.
  - 자막 위치(10등분)
  - 자막 스타일
  - 단어 하이라이트
  - 강조 강도
  - 키워드 강조
  - 컷별 키워드 사용
  - 등장 애니메이션(6가지 모션 포함)

## 커스텀 이미지 덮어쓰기

앱 화면에서 각 카드의 `이미지 선택`으로 업로드하면 해당 템플릿 미리보기 이미지를 사용자 버전으로 덮어쓸 수 있습니다.

- 저장 위치: 브라우저 로컬스토리지
- `이미지 초기화` 클릭 시 폴더 기본 이미지로 복귀
